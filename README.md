# LIT 공식 사이트

**[공식 사이트 바로가기](https://lhd-lit.github.io)**

LIT (Literature in Together)의 공식 웹사이트입니다

## 소개

**LIT (Literature in Together)**은 영어 문학 학습을 위한 AI 기반 학습 애플리케이션입니다. 개인 학습과 그룹 협업을 통해 문학 작품을 더 깊이 이해할 수 있도록 도와줍니다.

### 주요 특징

- **Focusing (AI Assistant)**: 개인 맞춤형 AI 학습 어시스턴트를 통한 깊이 있는 문학 작품 학습
- **Brain Storming**: 그룹 스터디를 통한 협업 학습 및 토론

### 지원 플랫폼

- Windows 10 이상
- macOS 10.13 이상 (Intel & Apple Silicon)
